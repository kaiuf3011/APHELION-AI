# APHELION AI UI Component System

Generate a cohesive collection of premium React UI components for a scientific mission-control dashboard called **APHELION AI**.

The application is an AI-powered Solar Flare Forecasting & Nowcasting platform for ISRO's Aditya-L1 mission.

This is NOT a generic admin dashboard.

The design should feel like software used inside:

- NASA Mission Control
- ESA Space Weather Center
- ISRO Mission Operations
- Bloomberg Terminal
- Palantir Foundry

The components must share one consistent design language.
